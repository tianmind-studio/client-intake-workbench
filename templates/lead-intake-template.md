# Lead Intake Template

Use this template to decide whether a lead is worth taking, what to quote, and what information is missing.

## Basic Info

```text
Client / source:
Contact:
Channel:
Received at:
Deadline:
Budget range:
Rush request:
```

## Original Message

Keep the important original wording in your private system. Use synthetic text in public examples.

```text
Original message:
>
```

## Project Type

- [ ] Website / dashboard
- [ ] Mini program
- [ ] App
- [ ] Python / data / visualization
- [ ] Java / PHP / Node project
- [ ] Deployment
- [ ] Document / delivery package
- [ ] Bug fix
- [ ] Other:

## Existing Materials

- [ ] Source code
- [ ] Run instructions
- [ ] Database
- [ ] Screenshots
- [ ] Account access
- [ ] Domain / server
- [ ] Platform / school requirements
- [ ] Reference case

## Missing Information

```text
Questions to ask:
1.
2.
3.
```

## First Judgment

```text
Can take this work:
Reason:
Risk:
Suggested quote:
Suggested delivery time:
After-sales boundary:
```

## First Reply Draft

```text
I will first check whether this can be handled. Please send the source zip, run instructions, database file, and the final expected result. If a server or domain is involved, include that too. After reviewing the materials, I will confirm the quote and timeline so the scope stays clear.
```
